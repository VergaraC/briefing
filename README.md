# Briefing

A fast, free, personal news reader. One static page, no backend, no accounts.

- ~25 RSS sources across World / Tech & AI / Startups / Business / Brasil tabs
- Smart "Top" ranking: freshness × multi-outlet coverage × your reading taste (learned on-device)
- Story clusters collapse into one card with links to other outlets' coverage
- Expandable summaries, search, native share, read-story dimming, offline support
- Headlines fetched live from the sources' RSS feeds via CORS relays (`worker.js` is an optional self-hosted relay for Cloudflare Workers)

Open it, add to your phone's home screen, done.
